What is CSS (Curvature Scale Space)?

CSS is a powerful multi-scale shape representation technique based on the evolution of curvature zero-crossings as a planar curve is smoothed with increasing Gaussian filters. This method is invariant to rotation, translation, and scale, making it ideal for shape recognition and matching.
